# Exploratory Data Analysis (EDA) Project

## Project Overview

This project performs Exploratory Data Analysis (EDA) on student performance data to identify trends, correlations, and influencing factors.

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn

---

## Features

- Statistical summaries
- Correlation analysis
- Data visualization
- Trend identification
- Insight generation

---

## Visualizations Included

- Gender Distribution
- Math Score Distribution
- Correlation Heatmap
- Parental Education vs Performance

---

## Key Insights

- Reading and writing scores are strongly correlated.
- Students with educated parents tend to score better.
- Most students scored in the average range.

---

## How to Run

Install required libraries:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
