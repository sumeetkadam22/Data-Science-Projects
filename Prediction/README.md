# Predictive Modeling Using Machine Learning

## Project Overview

This project demonstrates predictive modeling using Machine Learning algorithms.

The project predicts house prices using the California Housing Dataset and compares the performance of multiple supervised learning models.

---

## Technologies Used

- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## Machine Learning Algorithms Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

---

## Features

- Data preprocessing
- Model training and testing
- Performance evaluation
- Visualization of predictions
- Feature importance analysis

---

## Evaluation Metrics

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R2 Score

---

## Visualizations

- Actual vs Predicted Prices
- Feature Importance Graph

---

## How to Run

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
python main.py
```
